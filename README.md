# 🌸 Calculator

A stylish, aesthetic web-based calculator built using **HTML, CSS, and JavaScript**.

This project combines functionality with a soft pink premium UI design, smooth button effects, and a sliding history panel.

---

## ✨ Features

- 🧮 Basic arithmetic operations (+, −, ×, ÷)
- 📊 Calculation history panel
- 🧠 Memory functions (M+, M-, RM)
- 💾 History saved using localStorage
- 🎨 Premium pink 3D UI design
- 🔊 Button click sound effect
- 📱 Responsive layout
- 💫 Smooth animations and hover effects

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation & localStorage)
